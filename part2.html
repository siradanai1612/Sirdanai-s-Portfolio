หน้านี้ใช้เเสดงประสบการณ์และผลงาน

ประสบการณ์

ผมได้มีโอกาสทำงานที่ร้านเอกสารและไอที ซึ่งทำให้ผมได้เรียนรู้การทำงานเป็นทีม ระบบการจัดการของบริษัท รวมถึงด้านไอที เช่น การซ่อมคอมพิวเตอร์เบื้องต้น การทำคลิปวิดีโอลงในสื่อโซเชียล การตัดต่อวิดีโอ การสื่อสารกับบุคคลภายนอก และการเขียนเว็บไซต์ ผมยังได้มีโอกาสศึกษาเกี่ยวกับการเขียนเว็บไซต์สำหรับร้านอาหาร ร้านสกรีนเสื้อ และร้านถ่ายเอกสารและไอทีที่ผมทำอยู่

ทักษะ

  • สามารถใช้ Photoshop ในระดับพื้นฐาน
  • สามารถใช้ Premiere Pro ในระดับพื้นฐาน
  • สามารถใช้ HTML code ในระดับพื้นฐาน
  • สามารถใช้ CSS code ในระดับพื้นฐาน
  • สามารถใช้ JavaScript code ในระดับพื้นฐาน
    • สามารถใช้ได้ในขั้นพื้นฐาน
